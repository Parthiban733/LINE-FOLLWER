# Line Follower using Arduino UNO with 5channel IR sensor 

# Required:
    1.Arduino UNO 
    2.BFD-1000 Five Channel Infrared Tracking Sensor Module 
    3.L298N 2A Based Motor Driver Module
    4.Two toy motors
    5.2 toy car Tyres
